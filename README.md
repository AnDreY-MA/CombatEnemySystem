# Combat Enemy System

A plugin containing base classes for creating combat AI with GAS integration, and using StateTree

## Demonstration

### Features:
- melee and ranged attacks
- detection sight, noise
- running away from granade
- take cover, firing from cover
- reloading in safe place, whene magazine is empty 
- throw granade, when target taking cover is long time
- patroling
- Notification to allies about target/enemy
